# supply-chain-anomaly-detection
Supply Chain Anomaly Detection using Machine Learning | Detects anomalies in supply chain data using Isolation Forest, Autoencoder & other ML models. Detecting anomalies in supply chain operations using Unsupervised (Isolation Forest, LOF, One-Class SVM) and Supervised (Random Forest, Gradient Boosting, Logistic Regression, SVM) ML models | Features: lead time, supplier reliability, defect rate, shipment delay & more | Evaluated using ROC-AUC, F1-Score, Precision-Recall Curves
